# Java

Mohist `1.7.10` - `1.12.2` requires **[Java 8](https://adoptium.net/?variant=openjdk8&jvmVariant=hotspot)**.

Mohist `1.16.5` requires **[Java 11](https://adoptium.net/?variant=openjdk11&jvmVariant=hotspot)** or higher, recommended (**Java 11**).

Mohist `1.18.2` - `1.20.2` requires **[Java 17](https://adoptium.net/?variant=openjdk17&jvmVariant=hotspot)** or higher, recommended (**Java 17**).

Mohist `1.21.1+` requires **[Java 21](https://adoptium.net/?variant=openjdk21&jvmVariant=hotspot)** or higher, recommended (**Java 21**).

Need more Java versions? Click [here](https://adoptium.net/).
