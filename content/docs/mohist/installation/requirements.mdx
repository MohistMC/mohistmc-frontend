# Java

Mohist `1.7.10` - `1.12.2` 需要 **[Java 8](https://adoptium.net/?variant=openjdk8&jvmVariant=hotspot)**。

Mohist `1.16.5` 需要 **[Java 11](https://adoptium.net/?variant=openjdk11&jvmVariant=hotspot)** 及以上, 推荐(**Java 11**)。

Mohist `1.18.2` - `1.20.2` 需要 **[Java 17](https://adoptium.net/?variant=openjdk17&jvmVariant=hotspot)** 及以上, 推荐(**Java 17**)。

Mohist `1.21.1+` 需要 **[Java 21](https://adoptium.net/?variant=openjdk21&jvmVariant=hotspot)** 及以上, 推荐(**Java 21**)。

需要更多版本的 Java？点击 [这里](https://adoptium.net/)。
