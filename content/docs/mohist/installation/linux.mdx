### Download

<p>To create a Mohist server, you need to download from the <a className="link link-info" href="/download/mohist" target="_blank">official download</a></p>

<p>You can also download via CLI:</p>

<div className="tabs tabs-lift">
    <input type="radio" name="my_tabs_3" className="tab" aria-label="wget" defaultChecked/>
    <div className="tab-content bg-base-100 border-base-300 p-6">
        ```bash
        wget https://api.mohistmc.com/project/{projectName}/{versionName}/builds/{buildId}/download
        ```
    </div>

    <input type="radio" name="my_tabs_3" className="tab" aria-label="curl" />
    <div className="tab-content bg-base-100 border-base-300 p-6">
        ```bash
        curl -O https://api.mohistmc.com/project/{projectName}/{versionName}/builds/{buildId}/download
        ```
    </div>
</div>

<p>Make sure you download the correct version you need. Please refer to <a href="/docs/mohist/versions">minecraftVersion</a> to download the version you need. Please click
    <a className="link link-info" href="https://mohistmc.cn/download/mohist">Available Versions</a> for more information.</p>

### Create Folder

<p>Create a folder with the server name of your choice.
    After creating the new folder, move the downloaded Mohist file to this new folder.</p>

<div className="alert alert-info my-4">
    <svg xmlns="http://www.w3.org/2000/svg" className="stroke-current shrink-0 h-6 w-6" fill="none" viewBox="0 0 24 24"><path strokeLinecap="round" strokeLinejoin="round" strokeWidth="2" d="M13 16h-1v-4h-1m1-4h.01M21 12a9 9 0 11-18 0 9 9 0 0118 0z" /></svg>
    <span>If you already have a Spigot/Forge server and want to switch to Mohist, it's better to create a new server and then add your world, mods, and plugins.</span>
</div>

### Run Server

<div className="tabs tabs-lift">
    <input type="radio" name="runTabs" className="tab" aria-label="Script" defaultChecked />
    <div className="tab-content bg-base-100 border-base-300 p-6">
        You must create a script file named `start.sh` in your server directory

        Write this content to the `start.sh` file:
        ```bash
        java -jar -Xmx4G downloaded-jar-file.jar
        ```

        Replace `downloaded-jar-file.jar` with the actual name of the jar file you dragged into the server folder.

        After completion, execute the script `/start.sh`
    </div>

    <input type="radio" name="runTabs" className="tab" aria-label="Terminal" />
    <div className="tab-content bg-base-100 border-base-300 p-6">
        Open a terminal and navigate to your server folder.

        Then, run the following command:
        ```bash
        java -jar -Xmx4G downloaded-jar-file.jar
        ```

        Replace `downloaded-jar-file.jar` with the actual name of the jar file you dragged into the server folder.
    </div>
</div>
