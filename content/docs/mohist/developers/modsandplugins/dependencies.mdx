title: "Dependencies"

You may want to access MohistAPI in your plugins or mods, this page is for you. It is recommended to use the official template for plugin development.
Recommended use of [official template](https://github.com/MohistMC/MohistPlugin) Suggested use.

### 1.12.2

```groovy
repositories {
    maven {url ""https://maven.mohistmc.com/""}
    maven {url ''https://hub.spigotmc.org/nexus/content/groups/public/''}
    maven {url ''https://libraries.minecraft.net/''}
    maven {url ''https://maven.minecraftforge.net/''}
}

dependencies {
    implementation ''com.mohistmc:mohistdev:0.1-SNAPSHOT''
}
```
