### 🎨 Core features:
* Global staining - automatically applied everywhere (to be improved)
* Multi-format compatibility - supports & symbol, hexadecimal, gradient, solid color, mixed formats、[中国传统色](/tools/ctc)
* Intelligent parsing - automatically recognizes and converts color codes
* Seamless integration - Other plugins can be used without modifications

### ✨ Application Scenarios
* Message staining - chat, broadcast messages
* Interface beautification - menus, titles, scoreboard
* Ambient decoration - signs, book text
* Dynamic effects - gradient colors, special formats

### 🛠️ Usage examples

#### During the development process ↓ There is no need to write additional formatting analysis by yourself
```java
// 1. Gradient format (with and without # supported)
sender.sendMessage("<gradient:#FF0000:#0000FF>Gradient text</gradient>");
sender.sendMessage("<gradient:FF0000:0000FF>Gradient text</gradient>");
sender.sendMessage("<gradient:#00FF00:#0000FF>Green to blue gradient</gradient>");

// 2. Fixed color format (with and without # supported)
sender.sendMessage("<#FF0000>Red text</#FF0000>");
sender.sendMessage("<FF0000>Red text</FF0000>");
sender.sendMessage("<red>Red text</red>");
sender.sendMessage("<bold>Bold text</bold>");
sender.sendMessage("<月白>月白色文本</月白>");

// 3. Hex format
sender.sendMessage("&#FF0000Red text\n<gradient:#00FF00:#0000FF>Green to blue gradient</gradient>");

// 4. Traditional format
sender.sendMessage("&agreen &cred &lbold");

// 5. Mixed format
sender.sendMessage(
        "<gradient:#FF0000:#00FF00>Gradient text</gradient> " +
        "<#0000FF>Blue text</#0000FF> " +
        "&#FF0000Red text " +
        "&aGreen text"
);

// 6. Examples of other methods
Bukkit.broadcastMessage("<gradient:#00FF00:#0000FF>This is a green-to-blue gradient announcement</gradient>");
player.setDisplayName("<#FF0000> playername</#FF0000>");
For more ways to play, please explore by yourself
```
