title: "DeepSeek"

`youer-config/youer.yml`
```yaml
deepseek:
  # 是否启用DeepSeek功能
  enable: false
  # DeepSeek API的请求地址
  baseUrl: "https://api.deepseek.com/chat/completions"
  # API密钥，用于身份验证。 请自行申请并确保有余额~
  apikey: "youer"
  # 使用的模型名称 可用的模型名称：deepseek-chat、deepseek-reasoner
  model: "deepseek-chat"
  # 系统提示词，定义AI助手的人设
  system: "你的名字叫小小墨，年龄18岁，是个可爱的女孩子!"
  # 触发AI对话的命令(私聊) 用法: ai [内容]
  command: "ai"
  # 触发AI对话的命令(全服可见) 用法: ai-all [内容]
  all_command: "ai-all"
  # 聊天消息的格式，%s会被替换为实际回复内容
  chatformat: "<小小墨> %s"
```
