### How to use Velocity with Youer?

-> **paper-config/paper-global.yml**
```yaml
proxies:
  velocity:
    enabled: true
    online-mode: false
    secret: ''
```

-> **server.properties**

```properties
online-mode=false //If you still want a premium-only server, you can set it later in Velocity's configuration file
```

- Now, create a new folder for your Velocity proxy and start it with the following java command:

```bash
java -jar [velocity jar name].jar
```

- Then after the proxy is fully started, type "**end**" to shut down your proxy.
- Now, you need to modify **velocity.toml** in the Velocity folder:

-> **velocity.toml**

```toml
player-info-forwarding-mode = "modern"
```

For more detailed tutorials, please read the <a className="link link-info" href="https://docs.papermc.io/paper/" target="_blank">Paper documentation</a> and <a className="link link-info" href="https://docs.papermc.io/velocity" target="_blank">Velocity documentation</a>

- Now you can start your proxy and all Youer servers!

### ‚ùì FAQ
- **Q:** Cannot enter lobbies made with Paper and other vanilla servers?
- **A:** Install the <a className="link link-info" href="https://modrinth.com/mod/be-quiet-negotiator" target="_blank">Be Quiet Negotiator</a> mod on the client + install the <a className="link link-info" href="https://github.com/MohistMC/ModPacketFix" target="_blank">ModPacketFix</a> plugin on the Paper server to resolve.
- **Q:** Client encounters `Network Protocol Error`?
- **A:** Add the parameter ```-Dvelocity.max-known-packs=512``` to Velocity's startup parameters, replace ```512``` with the number you need (>64) according to your requirements.
