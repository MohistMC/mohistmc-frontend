#### Basic Usage:
`/youer packetstats [start|stop|status]`

#### Permission:
`youer.command.youer`

### -> start
`/youer packetstats start`
Start the analysis process
<img src="/img/docs/packetstats_start.webp" alt="Analysis Start" />

### -> status
`/youer packetstats status`
View the analysis process status
<img src="/img/docs/packetstats_status.webp" alt="Analysis Process" />

### -> stop
`/youer packetstats stop`
End the analysis process and display results
<img src="/img/docs/packetstats_stop.webp" alt="Analysis Results" />
