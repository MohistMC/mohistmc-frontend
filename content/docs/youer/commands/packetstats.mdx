#### 基本用法:
`/youer packetstats [start|stop|status]`

#### 权限:
`youer.command.youer`

### -> start
`/youer packetstats start`
开启分析进程

### -> status
`/youer packetstats status`
查看分析进程状态
<img src="/img/docs/packetstats_status.webp" alt="分析过程" />

### -> stop
`/youer packetstats stop`
结束分析进程，并显示结果
<img src="/img/docs/packetstats_stop.webp" alt="分析结果" />
