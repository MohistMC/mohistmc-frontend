#### Basic Usage:
`/items [name|save|list|get|remove|modeldata|lore]`

#### Permission:
`youer.command.items`

### -> name
`/items name <string>`
Set the display name of the item in hand

### -> save
`/items save <name>`
Save the item in hand to configuration file for later use

### -> list
`/items list`
Open GUI interface to display all saved items

### -> get
`/items get <name>`
Get item from configuration file and put it in hand

### -> remove
`/items remove <name>`
Remove item from configuration file

### -> modeldata
`/items modeldata <int>`
Set the model data of the item

### -> lore add
`/items lore add <line> <text...>    # Insert after specified line`

Add lore to the item, supports color codes (&) and line breaks (\n)
* When line is not specified, add at the end
* When line is specified, insert after the specified line

### -> lore set
`/items lore set <line> <text...>`
Set the lore content of the specified line

### -> lore remove
`/items lore remove <line>`
Remove lore of the specified line

### -> attribute
`/items attribute <attribute> <value> [slot]`
Add attribute modifier to the item
* attribute: Attribute name (e.g. GENERIC_MAX_HEALTH or MAX_HEALTH)
* value: Attribute value (number)
* slot: Equipment slot (HAND, OFF_HAND, FEET, LEGS, CHEST, HEAD, ANY)

### -> unattribute
`/items unattribute <attribute>`
Remove the specified attribute modifier from the item
