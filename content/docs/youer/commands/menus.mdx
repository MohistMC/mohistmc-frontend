#### Basic Usage:
`/menus open menu_filename` with Tab auto-completion

[Companion Menu Editor](/tools/menu-edit)

## Menu Basic Settings

`menu-settings` basic properties:

- `name`: Menu title display text (supports color codes)
- `rows`: Menu rows (1-6 rows)
- `open-actions`: Actions executed when opening the menu

## Icon Configuration Description

Each icon item includes the following optional properties:

### Basic Properties
- `MATERIAL`: Item material (supports Minecraft item IDs)
- `POSITION-X`: X coordinate position (1-9)
- `POSITION-Y`: Y coordinate position (1-6)
- `NAME`: Display name (supports color codes)
- `LORE`: Description text list (supports color codes)

### Functional Properties
- `ACTIONS`: List of actions executed on click
- `DURABILITY`: Durability (for displaying damage effect)
- `ENCHANTMENTS`: Enchantment effect list
- `ITEMFLAG`: Item flags (such as HIDE_ENCHANTS)
- `AMOUNT`: Item amount
- `PERMISSION`: Required permission node
- `PERMISSION-MESSAGE`: Message displayed when no permission
- `CUSTOMMODELDATA`: Custom model data
- `HIDE-TOOLTIP`: Whether to hide tooltip information
- `KEEP-OPEN`: Whether to keep menu open after clicking
- `BASE64`: Custom skin Base64 for player heads

### Action Types
- `tell: <message>`: Send message to player
- `console: <command>`: Execute command as console
- `op: <command>`: Execute command as OP
- `player: <command>`: Execute command as regular player
