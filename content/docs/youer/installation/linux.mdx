### Download

To create a Youer server, you need to download from the <a className="link link-info" href="/download/youer" target="_blank">official download</a>

### Create Folder

Create a folder with the server name of your choice.
    After creating the new folder, move the downloaded Youer file into this new folder.

### Run the Server

<div role="tablist" className="tabs tabs-lifted my-6">
    <input type="radio" name="runTabs" role="tab" className="tab" aria-label="Script" defaultChecked />
    <div role="tabpanel" className="tab-content bg-base-100 border-base-300 rounded-box p-6">
        You must create a script file named `start.sh` in your server directory

        Write this content to the `start.sh` file:
        ```text
        java -jar -Xmx4G downloaded-jar-file.jar
        ```
        Replace `downloaded-jar-file.jar` with the actual name of the jar file you dragged into the server folder.

        After that, execute the script with `./start.sh`
    </div>
