### Download

<p>To create a Youer server, you need to download from the <a className="link link-info" href="/download/youer" target="_blank">official download</a></p>

### Create Folder

<p>Create a folder with the server name of your choice.
    After creating the new folder, move the downloaded Youer file into this new folder.</p>

<div className="alert alert-info my-4">
    <svg xmlns="http://www.w3.org/2000/svg" className="stroke-current shrink-0 h-6 w-6" fill="none" viewBox="0 0 24 24"><path strokeLinecap="round" strokeLinejoin="round" strokeWidth="2" d="M13 16h-1v-4h-1m1-4h.01M21 12a9 9 0 11-18 0 9 9 0 0118 0z" /></svg>
    <span>If you already have a Spigot/Forge server and want to switch to Youer, it's better to create a new server and then add your worlds, mods, and plugins.</span>
</div>

### Run the Server

<p>Almost done! To run your server, do not double-click the <strong>jar</strong> file! You must create a file named `start.bat` where you can place your Youer jar file.</p>

<p>Write the following content to the `start.bat` file:</p>
```text
java -jar -Xmx4G downloaded-jar-file.jar
```
<p>Replace `downloaded-jar-file.jar` with the actual name of the jar file you dragged into the server folder.</p>

<div className="alert alert-warning my-4">
    <svg xmlns="http://www.w3.org/2000/svg" className="stroke-current shrink-0 h-6 w-6" fill="none" viewBox="0 0 24 24"><path strokeLinecap="round" strokeLinejoin="round" strokeWidth="2" d="M12 9v2m0 4h.01m-6.938 4h13.856c1.54 0 2.502-1.667 1.732-3L13.732 4c-.77-1.333-2.694-1.333-3.464 0L3.34 16c-.77 1.333.192 3 1.732 3z" /></svg>
    <span>Make sure the server filename in `start.bat` ends with `jar`</span>
</div>

<p>When finished, double-click `start.bat`</p>
