### 经过一个月的测试，Youer-1.21.1正式版在2025年6月1日正式发布。

Youer是 NeoForge + Paper API的混合服务器软件
* 关于制作进度可以查看
    * <a className="link link-warning" href="https://github.com/MohistMC/Youer/issues/1" target="_blank">Youer-1.21.1-进度</a>
    * 目前的进度并没有合并制作Paper API，所以目前Youer-1.21.1仅支持NeoForge + Spigot API，会在后续的更新中支持Paper API
* 内置的功能大部分和Mohist-1.20.1一致
    * 世界管理器已支持超平坦世界的创建
    * 更多内容可查看后续制作的文档
* 兼容性
    * 将优先处理mod的兼容修复，目前无法保证兼容所有mod
    * 插件的兼容性目前无法保证完美，且优先级比较低，更专注于mod的兼容
* 反馈途径
    * 提交Github issue
    * Discord #Donator角色 优先频道
* 内测用户
    * 测试人员可以提交Github issue(优先处理权)，或者加入会员专群
    * 提前使用已修复但未发布的版本(测试版本)
* 版本更新频率
    * 正式版本更新频率为每星期一次
    * 测试版本更新频率为每日不定时更新

### FAQ
为什么每周才发布一次更新，并采用内测用户的测试版？
 * 因为开发需要成本，且我们同时维护了多个框架的服务器软件，所以只能采取这种策略。

为什么我反馈的问题没有被处理？
 * 因为我们没有足够的时间处理所有的问题，所以我们只能选择处理优先级高的问题。同时也存在技术难度问题，所以不能保证所有问题都能被解决。

Youer是否会制作更多版本？
 * 目前仅提供1.21.1版本，后续版本目前没有考虑。除非你愿意等待1.22.1。
