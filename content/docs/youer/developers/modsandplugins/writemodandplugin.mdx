## 环境介绍

Youer 旨在支持 NeoForge 模组 和 Bukkit/Spigot/Paper 插件。

### 如何为 Youer 编写 插件?

由于 Youer 基于 NeoForge 并实现了 Paper API，因此您可以完全按照[Paper的官方文档](https://docs.papermc.io/paper/)进行编写, 无需额外步骤。

如果你不需要 Paper API或者考虑兼容原版Spigot服，你可以按照[Spigot的官方文档](https://www.spigotmc.org/wiki/spigot-plugin-development/)进行编写。

Youer 提供了一些 便捷 API 供你使用。Youer继承了mohist的api，不过因为版本跨度太大，需要重新设计，请等待后续的介绍！
