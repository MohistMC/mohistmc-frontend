# Write mods and plugins for Youer

Youer is built to support NeoForge mods and Bukkit/Spigot/Paper/Purpur plugins.

## How to write mods and plugins for Youer?

Since Youer is based on Forge and implements Bukkit/Paper API, you can write mods and plugins for Youer just like you
write mods and plugins for Forge and Bukkit/Paper, no extra steps needed.

<Callout>
    Remember that Youer is still in development, and some features of some
    plugins may not work properly.
</Callout>

### Youer API

Youer provides some APIs for mods and plugins that you can use. You can find them in the [API Reference](dependencies.mdx).
