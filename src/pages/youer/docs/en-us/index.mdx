# Introduction to Youer

Youer is a revolutionary Minecraft server core based on NeoForge and fully compatible with Bukkit/Spigot/Paper API. It breaks the barriers between the module and plugin ecosystem, allowing server administrators to take advantage of the powerful functions of Forge modules and the efficient ecosystem of Bukkit plugins without compromise.

## Quick Start

To get started with Youer, you need to follow the [requirements](/youer/docs/en-us/requirements).
