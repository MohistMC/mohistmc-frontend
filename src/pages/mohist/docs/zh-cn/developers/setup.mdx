# 设置 Mohist 进行开发

<Steps>
    ### 克隆项目

    ```bash
    git clone https://github.com/MohistMC/Mohist.git -b mc-version
    cd Mohist
    ```

    或者手动将其克隆到空文件夹。
    ### 设置工作区

    <Tabs items={['Windows', 'Linux']}>
        <Tabs.Tab>
            ```bash
            gradlew.bat setup
            ```
        </Tabs.Tab>
        <Tabs.Tab>
            ```bash
            ./gradlew setup
            ```
        </Tabs.Tab>
    </Tabs>

</Steps>
