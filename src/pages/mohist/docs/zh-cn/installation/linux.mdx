# 在 Linux 上安装 Mohist

<Steps>
    ### 下载

    要创建 Mohist 服务器,您需要从[官方下载](https://mohistmc.com/downloadSoftware?project=mohist).

    您也可以通过 CLI 下载:
    <Tabs items={['wget', 'curl']}>
        <Tabs.Tab>
            ```sh
            wget https://mohistmc.com/api/v2/projects/mohist/[minecraftVersion]/builds/latest/download
            ```
        </Tabs.Tab>
        <Tabs.Tab>
            ```sh
            curl -O https://mohistmc.com/api/v2/projects/mohist/[minecraftVersion]/builds/latest/download
            ```
        </Tabs.Tab>
    </Tabs>

    确保您下载到需要的正确版本。 请看参考 [minecraftVersion](/mohist/docs/versions) 下载您所需要的版本。 请点击
    [可用版本](https://mohistmc.com/downloadSoftware?project=mohist) 了解更多信息。

    ### 创建文件夹

    使用您选择的服务器名称创建一个文件夹。
    新建文件夹完成后，将下载的Mohist文件移动到这个新的文件夹里面。

    <Callout>
        如果您已经有 Spigot / Forge 服务器并想切换到 Mohist，最好创建一个新的 服务器，然后添加您的地图、模组和插件。
    </Callout>

    ### 运行服务器

    快完成了！要运行您的服务器, **不要双击 jar 文件！**

    <Tabs items={['使用脚本', '使用终端']}>
        <Tabs.Tab>
            您必须创建一个名为
            `start.sh` 您放置 Mohist jar 文件的位置

            将此内容写入 `start.sh` 文件：
            ```bash
            java -jar -Xmx4G downloaded-jar-file.jar
            ```

            将 `downloaded-jar-file.jar` 替换为您拖放到服务器文件夹的 jar 文件的真实名称。

            完成后，双击 `start.sh`
        </Tabs.Tab>
        <Tabs.Tab>
            打开一个终端，然后转到您放置 Mohist jar 文件的文件夹。

            然后，运行以下命令：
            ```bash
            java -jar -Xmx4G downloaded-jar-file.jar
            ```
            替换为您拖放到服务器文件夹的 jar 文件的真实名称。downloaded-jar-file.jar
        </Tabs.Tab>
    </Tabs>

</Steps>
