# 在 Windows 上安装 Mohist

<Steps>
    ### 下载

    要创建 Mohist 服务器，您必须从 [官方下载](https://mohistmc.com/downloadSoftware?project=mohist)

    ### 创建文件夹

      使用您选择的服务器名称创建一个文件夹。
      新建文件夹完成后，将下载的Mohist文件移动到这个新的文件夹里面。

    <Callout>
         如果您已经有 Spigot / Forge 服务器并想切换到 Mohist，最好创建一个新的 服务器，然后添加您的地图、模组和插件。
    </Callout>

    ### 运行服务器

    快完成了！要运行您的服务器，请不要双击 **jar** 文件！ 您必须创建一个名为 `start.bat` 的文件，您可以在其中放置 Mohist jar 文件。

    将以下内容写入 `start.bat` 文件：
    ```bash
    java -jar -Xmx4G downloaded-jar-file.jar
    ```

    将 `downloaded-jar-file.jar` 替换为您拖放到服务器文件夹的 jar 文件的真实名称。

    <Callout type="warning">确保 `start.bat` 中的服务器文件名以 `jar` 结尾</Callout>

完成后，双击 `start.bat`

</Steps>
