**Mohist** is a Minecraft Forge server software which implements Bukkit/Spigot/Paper APIs. This software lets you combine
the best of both worlds: the power of Forge and Bukkit ecosystem.

## Quick start

To start using Mohist, you need to follow the [requirements](/mohist/docs/requirements).
