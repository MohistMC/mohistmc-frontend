# Java

Mohist `1.7.10` and `1.12.2` requires **[Java 8](https://adoptium.net/?variant=openjdk8&jvmVariant=hotspot)**.

Mohist `1.16.5` requires Java between **[Java 11](https://adoptium.net/?variant=openjdk11&jvmVariant=hotspot)** and  **[Java 16](https://adoptium.net/?variant=openjdk16&jvmVariant=hotspot)** (**Java 11** is recommended).

Mohist `1.18.2` to `1.20.1` requires **[Java 17](https://adoptium.net/?variant=openjdk17&jvmVariant=hotspot)** and above (**Java 17** is recommended).

Need more versions of java? Click [here](https://adoptium.net/).
