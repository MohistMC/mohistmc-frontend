# Build Mohist with Gradle

<Callout type="warning">Don't forget to [generate patches](patches.mdx) if you modified Minecraft source code.</Callout>

<Tabs items={['1.7.10', '1.12.2', '1.16.5+']}>
    <Tabs.Tab>
        Build with Linux:
        ```bash
        bash gradlew setupCauldron launch4j
        ```

        Build with Windows:
        ```bash
        gradlew.bat setupCauldron launch4j
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        Build with Linux:
        ```bash
        bash gradlew setup installerJar
        ```

        Build with Windows:
        ```bash
        gradlew.bat setup installerJar
        ```
    </Tabs.Tab>
    <Tabs.Tab>
        Build with Linux:
        ```bash
        bash gradlew setup mohistJar
        ```

        Build with Windows:
        ```bash
        gradlew.bat setup mohistJar
        ```
    </Tabs.Tab>
</Tabs>

<Callout type="info">The Mohist server jar file is located at ./projects/mohist/build/libs/mohist-xxxxx-server.jar</Callout>

This is the **jar file** that you will need to run your server.
